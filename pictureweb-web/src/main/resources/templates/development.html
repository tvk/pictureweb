Some information about the build, deployment and infrastructure of this website:

Infrastructure

Let's Encrypt

This great service provides this website with free ssl support!

Eureka

Eureka is the service discovery tool. As this application is built in microservices we it needs a service discovery tool where all microservices register themselves and can discover other microservices that they need to run.

Jenkins

The well known build tool jenkins is used to build all microservices of this application after each commit.
